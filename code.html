Dig It All

Dig it all

Dig It All is a metal detecting game I made. The link is to a zipfile from my github. You will have to unzip it to play the game. Once you extract the files, click into the folder and run the digItAll_v2.exe file. It will only work on Windows, though I do have a Mac version, it is not uploaded. All images used in the game are real coins or relics that either Stef or I have found metal detecting.

Pig Game

Simple Question Game
